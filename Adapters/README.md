# Adapters

## Overview

Adapters enable communication between SAP Integration Suite and external systems by supporting various communication protocols.

---

## Adapters Covered

### Sender Adapters

- HTTP
- HTTPS
- SOAP
- SFTP
- IDoc
- OData
- JMS
- Mail

### Receiver Adapters

- HTTP
- HTTPS
- SOAP
- SFTP
- IDoc
- OData
- SuccessFactors
- Ariba
- XI
- JDBC
- RFC

---

## Common Use Cases

- SAP to SAP Integration
- SAP to Non-SAP Integration
- Cloud Integration
- File Transfer
- API Communication
