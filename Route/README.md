# Routing Palette

## Overview

The Routing Palette controls the flow of messages within an Integration Flow based on conditions or processing requirements.

---

## Components Covered

- Router
- Filter
- Splitter
- Gather
- Join
- Multicast
- Sequential Multicast
- Aggregator

---

## Common Use Cases

- Conditional Routing
- Message Filtering
- Parallel Processing
- Sequential Processing
- Message Aggregation
