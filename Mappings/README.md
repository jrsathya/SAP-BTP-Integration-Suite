# Mappings

## Overview

Mapping is used to transform the source message into the target message by changing the structure, format, or values according to business requirements.

---

## Types of Mappings

- Message Mapping
- XSLT Mapping
- Groovy Mapping
- XML to JSON Mapping
- JSON to XML Mapping

---

## Common Use Cases

- Source to Target XML transformation
- Data conversion
- Field mapping
- Value Mapping
- Data enrichment
